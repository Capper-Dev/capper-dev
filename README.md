### Hey, I'm Casper

Self-taught developer from Denmark. I build tools and websites that solve real problems — or at least ones that annoy me enough to automate away.

Most of what I make lives in the React/Next.js ecosystem with TypeScript and Tailwind. I also spend time in the FiveM world building server scripts and UIs with Lua and React.

---

**Some things I've built:**

**[GitScope](https://github.com/Capper-Dev/gitscope)** — Turn any GitHub repo into an interactive visual graph. Force-directed layouts, treemaps, commit history, syntax-highlighted file viewer. Built because reading file trees is boring.

**[fivem-bundler](https://github.com/Capper-Dev/fivem-bundler)** — A build-time compiler for FiveM Lua resources. Resolves `require()` calls via AST analysis and outputs single-file bundles. Available on npm.

**[CFXFinder](https://github.com/Capper-Dev/cfxfinder)** — Server browser for the FiveM community.

---

I like building things that are useful, look good, and don't take themselves too seriously.
